# 家庭档案与设置页设计

日期：2026-04-13

## 目标

补齐家庭页档案展示、档案编辑、快捷设置，以及设置页的通知、提醒、图表和显示配置。V1 只保存配置，不接真实微信订阅消息，不做药物结构化管理，不做 UI 最终精修。

## 范围

本阶段包含：

- 家庭页被监护人档案展示。
- 家庭页档案编辑入口。
- 家庭页快捷设置。
- 设置页通知设置。
- 设置页提醒设置。
- 设置页图表设置。
- 设置页显示设置。
- 家庭成员处进入单个成员权限配置。

本阶段不包含：

- 真实微信订阅消息推送。
- 真实用药提醒推送。
- 药物结构化管理。
- 可编辑参考目标。
- 成员批量权限配置。
- 禁止查看家庭数据的权限。
- UI 最终精修。

## 家庭页档案

家庭页展示“被监护人档案”，并在档案区提供“编辑”入口。

展示字段：

- 姓名。
- 出生年。
- 年龄，按出生年计算。
- 当前血压，取最近一条记录，例如 `140 / 90 mmHg`。
- 当前心率，取最近一条记录，例如 `100 bpm`。
- 当前用药，自由文本，例如 `氨氯地平 5mg 每日一次`。
- 紧急联系人，格式为 `姓名 · 电话`。

可编辑字段：

- 姓名。
- 出生年。
- 当前用药。
- 紧急联系人姓名。
- 紧急联系人电话。

不可手动编辑字段：

- 当前血压。
- 当前心率。
- 年龄。

这些字段来自最近记录或出生年计算，不由用户手填。

## 家庭页快捷设置

家庭页下方增加“快捷设置”区域，方便老人或家属不进入深层设置页也能调整常用项。

快捷设置包含：

- 用药提醒：开 / 关。
  - 小字解释：`按晨/晚时间提醒`。
- 异常血压通知家人：开 / 关。
  - 小字解释：`超过阈值时通知已选择成员`。
- 字体大小：标准 / 大 / 超大。
  - 小字解释：`调整全局显示大小`。

V1 中“用药提醒”和“异常血压通知家人”只保存配置，不接真实微信订阅消息或真实提醒服务。

## 设置页

设置页保留偏系统和规则的配置，不放档案编辑主入口，不放成员权限主入口。

每个设置项都需要有一行小字解释，参考：

```text
主标题
说明小字
```

### 通知设置

字段：

- 异常血压通知：开 / 关。
  - 小字解释：`超过阈值时通知已选择成员`。
- 通知对象：点击选择具体成员。
  - 小字解释：`血压异常时通知谁`。
  - 默认不选，需要用户自己选择。
- 高压告警阈值。
  - 小字解释：`收缩压超过此值时触发通知`。
- 低压告警阈值。
  - 小字解释：`舒张压超过此值时触发通知`。

说明：

- V1 只保存配置，不接真实微信订阅消息。
- 如果未选择通知对象，即使开启异常血压通知，也只保存开关和阈值，不实际通知任何人。

### 提醒设置

字段：

- 用药提醒：开 / 关。
  - 小字解释：`每日定时提醒服药`。
- 晨提醒时间。
  - 小字解释：`早晨服药提醒时间`。
  - 默认 `08:00`。
- 晚提醒时间。
  - 小字解释：`晚上服药提醒时间`。
  - 默认 `20:00`。

说明：

- V1 只保存配置，不接真实提醒服务。
- 用药内容仍使用家庭档案中的“当前用药”自由文本。

### 图表设置

字段：

- 晨/晚测标注：开 / 关。
  - 小字解释：`录入时选择测量时段`。
- 早晚分线显示：开 / 关。
  - 小字解释：`图表中晨/晚测分两条线`。

说明：

- V1 先保存配置。
- 如果后续图表精修阶段未实现早晚分线，开关仍可先保存，不强制立即改变图表表现。

### 显示设置

字段：

- 字体大小：标准 / 大 / 超大。
  - 小字解释：`超大模式适合老人使用`。

说明：

- V1 保存配置，并优先影响家庭页、数据页、记录页、报告页的主要字号。
- 具体视觉精修放到后期统一处理。

## 成员权限

成员权限不放在设置页主内容里。

家庭成员加入后默认可以查看家庭健康记录，不做“可查看”开关。

从家庭页成员区域进入单个成员权限配置，针对每个成员单独设置：

- 可录入：允许新增记录。
- 可编辑：允许修改或删除记录。

不做一键批量设置，不做禁止查看家庭数据的权限。

权限说明文案：

```text
家人加入后默认可以查看家庭健康记录。
可录入：允许新增记录。
可编辑：允许修改或删除记录。
```

```text
abnormalBpNotifyEnabled false
notifyMemberIds []
alertSystolic 160
alertDiastolic 100

morningReminderTime 08:00
eveningReminderTime 20:00
morningEveningLabel false
splitLines false
fontSize standard
```

## 云函数

新增或调整云函数：

- `updateFamilySettings`：管理员保存 `families.profile` 和 `families.settings`。
- 现有 `updateMemberPermission`：继续只负责单个成员的 `canWrite` 和 `canEdit`。

权限规则：

- 只有 admin 可以编辑家庭档案、通知设置、提醒设置、图表设置、显示设置和成员权限。
- 普通成员可以查看家庭页档案和设置页内容，但不显示保存入口或不可编辑。
